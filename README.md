# Project-Phoenix
This a set of commands which could make any computer fly like Phoenix. I have used the name Phoenix because it is the bird which borns again from its ash just like my Compaq Presario CQ40 which has BORN again with the help of Linux. I have used these commands to fully optimise this laptop.
Reviving a 15-year-old Compaq Presario CQ40 (Single Core Celeron) using 426+ handcrafted Linux commands.

## Features:
- **Optimization:** ZRAM, aggressive service pruning, and thermal management (31°C idle).
- **UI/UX:** Full WhiteSur macOS-style interface on Zorin OS Lite.
- **Resilience:** Running on a dying HDD with 196 bad sectors!

Check the `.txt` files for the full command list. (Script automation coming soon!)
